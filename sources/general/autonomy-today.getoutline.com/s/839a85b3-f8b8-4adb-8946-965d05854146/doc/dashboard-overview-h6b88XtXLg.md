## #Pending board

- **Priority Pending** — higher-priority queue when configured.
 
- **Regular Pending** — standard pending calls.
 
- **Show return calls** — toggle return-call visibility when available.
 
- Empty state: **No pending calls**.
 

> Screenshot placeholder: Pending board with priority and regular sections

## #Sorting and density

- **Sort calls** — change sort order per tab.
 
- **Call list density** — compact vs comfortable card layout.
 

> Screenshot placeholder: Sort and density controls on a call board

## #Assigning units from the board

- Drag a unit from the **Active Units** sidebar onto a call card (when drag-and-drop is enabled).
 
- Alternatively open the call and use **Assign Unit** in call detail.
 

> Screenshot placeholder: Drag unit onto call card

## #Status timer alerts

When timers are due or overdue, a **Status Timer Alerts** overlay may appear globally (not only on the Status Timers tab).

> Screenshot placeholder: Status Timer Alerts popup

## #Recent tab search

- Quick search within Recent tab.
 
- **Advanced recent search** for more filters.
 
- **Find a Call (full history)** opens archive search — see [Find a call](https://./find-a-call.md).
 

> Screenshot placeholder: Recent tab with search field

## #Tips

- Use agency filter before comparing boards across departments.
 
- Fire/EMS tabs reduce noise when you dispatch by discipline.
 
- Refresh boards with **Reload Data** if WebSocket updates appear stuck.
 

## #Common mistakes

- Looking for closed calls only on Recent — use **Find a Call** for full history.
 
- Forgetting agency filter is on — boards look empty when wrong agencies selected.
