**DL**, **LP**, **NAME**, **AH**, **UH**, and **S** connect to your agency’s **CJIS / Commsys** setup when it is enabled.

- Include **state** when querying out-of-state plates or licenses.
- Results may open in a side panel, new window, or return asynchronously — behavior follows your integration.
- All queries should be **policy- and audit-compliant**; access is logged server-side.

If a query type is not enabled for your agency, use the full CJIS panel instead.
