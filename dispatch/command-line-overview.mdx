The **quick command line** is a fast, keyboard-first way to create calls, change unit status, send messages, and run queries without hunting through menus.

## Where it appears

- **Dashboard (global mode)** — Fixed bar at the bottom of the main dispatch screen. The badge shows **GLOBAL**.
- **Call detail (in-call mode)** — Bar at the bottom while viewing a single call. The badge shows the **CFS** number. Many commands omit the CFS argument because the call is implied.

## Opening the command line

- Click in the bar, or use the focus shortcut configured in **Dispatch CAD** (commonly **backtick** `` ` `` or **`/`** from the dashboard).
- Press **F1** or type **`HELP`** (or **`?`**) for a list of commands your agency allows. **Help matches where you are:** on **call detail**, the dialog includes extra **call-only** commands (status timer, persons, vehicles, report email, batch assign, etc.). On the **dashboard**, those are hidden and the footer reminds you to open a call to use them.

## After you run a command

Success and error messages appear as short on-screen notifications (toasts). Command output may also update the board via live data.
