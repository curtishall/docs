MDT settings beyond voice alerts and training mode.

## Voice and assignment alerts

**Voice Alerts** (Settings):

| Control | Effect |
|---------|--------|
| **Incoming call alerts** toggle | Speaks your custom phrase when a **new** call is assigned |
| **Custom phrase** | Text spoken for new assignments (e.g. “You have a call.”) |

When speech is off or the app is backgrounded, Android may show a **Call Assignments** notification instead. See [Voice alerts](/mobile/voice-alerts).

## Display mode

**SYSTEM CONFIGURATION** → **DISPLAY MODE**

| Mode | Use |
|------|-----|
| **Normal** | Standard colors |
| **Night Vision** | Reduced glare for night patrol |
| **Monitor Blackout** | Minimal screen output when required |

- Test printer after pairing before first citation stop.
- Run **CHECK UPDATES** after admin pushes new violation codes.
