Run person, vehicle, firearm, article, and boat queries from the mobile MDT.

<Note>Screenshot placeholder: NCIC Search screen with RESPONSES tab</Note>

## Open

Sidebar **NCIC Search**

If NCIC is not enabled for your agency or user, **NCIC Search** remains visible but dimmed. Tapping it shows:

`NCIC access is not enabled for either your agency or your user, contact your administrator`

While access is still loading, tapping may show a short “checking access” message instead.

Training mode enables NCIC practice queries without live CJIS access.

## Tabs

| Tab | Status |
|-----|--------|
| **RESPONSES** | Query result history |
| **PERSON** | Person query form |
| **VEHICLE** | Vehicle query form |
| **FIREARM** | Firearm serial query |
| **ARTICLE** | Article / serial number query |
| **BOAT** | Boat registration or hull ID query |

<Note>Screenshot placeholder: PERSON and VEHICLE query tabs</Note>

## Person query

**PERSON QUERY** fields:

- **LAST NAME**, **FIRST NAME**
- **DATE OF BIRTH**
- **OLN / DL NUMBER**

Validation: **Last Name or DL Number is required**

<Note>Screenshot placeholder: Person query form filled out</Note>

## Vehicle query

**VEHICLE QUERY** fields:

- **LICENSE PLATE**, **STATE**, **VIN**

Validation: **License Plate is required** (or provide VIN)

<Note>Screenshot placeholder: Vehicle query form</Note>

## Firearm query

**FIREARM QUERY** fields:

- **SERIAL NUMBER** (required)

## Article / serial query

**ARTICLE / SERIAL QUERY** fields:

- **SERIAL NUMBER** (required)
- **SERIAL NUMBER TYPE** (optional)

## Boat query

**BOAT QUERY** fields:

- **REGISTRATION NUMBER** and/or **HULL ID (HIN)** — at least one required

In training mode, serials or boat identifiers containing `STOLEN` return sample stolen hits; otherwise you get a no-record style response.

## Responses

Actions on results (availability varies):

- **ISSUE CITATION**, **ISSUE WARNING**, **CREATE FIR**
- **REFRESH**, **LISTEN**, **PRINT PACKET**, **VIEW DETAILS**

<Note>Screenshot placeholder: NCIC response card with action buttons</Note>

## Voice summaries

Enable **CJIS result summaries** in Settings — see [Voice alerts](/mobile/voice-alerts).

## In-call queries

Person/vehicle **QUERY** buttons also appear on call detail cards.

## Requirements

Agency **State/NCIC Access** and user authorization in Web Admin. Criminal history queries are not available on mobile.

## Warnings

<Warning>CJIS policy applies on mobile same as desktop — official use only.</Warning>
