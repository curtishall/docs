Run person and vehicle queries from the mobile MDT.

<Note>Screenshot placeholder: NCIC Search screen with RESPONSES tab</Note>

## Open

Sidebar **NCIC Search**

## Tabs

| Tab | Status |
|-----|--------|
| **RESPONSES** | Query result history |
| **PERSON** | Active person query form |
| **VEHICLE** | Active vehicle query form |
| **FIREARM** | Coming Soon (typical) |
| **ARTICLE** | Coming Soon |
| **BOAT** | Coming Soon |

<Note>Screenshot placeholder: PERSON and VEHICLE query tabs</Note>

## Person query

**PERSON QUERY** fields:

- **LAST NAME**, **FIRST NAME**
- **DATE OF BIRTH**
- **OLN / DL NUMBER**

Validation: **Last Name or DL Number is required**

<Note>Screenshot placeholder: Person query form filled out</Note>

## Vehicle query

**VEHICLE QUERY** fields:

- **LICENSE PLATE**, **STATE**, **VIN**

Validation: **License Plate is required**

<Note>Screenshot placeholder: Vehicle query form</Note>

## Responses

Actions on results (availability varies):

- **ISSUE CITATION**, **ISSUE WARNING**, **CREATE FIR**
- **REFRESH**, **LISTEN**, **PRINT PACKET**, **VIEW DETAILS**

<Note>Screenshot placeholder: NCIC response card with action buttons</Note>

## Voice summaries

Enable **CJIS result summaries** in Settings — see [Voice alerts](/mobile/voice-alerts).

## In-call queries

Person/vehicle **QUERY** buttons also appear on call detail cards.

## Requirements

Agency **State/NCIC Access** and user authorization in Web Admin.

## Warnings

<Warning>CJIS policy applies on mobile same as desktop — official use only.</Warning>

<Warning>**Coming Soon** tabs are not production queries — do not use for real hits.</Warning>
