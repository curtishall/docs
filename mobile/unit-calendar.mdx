Shift schedule, court dates, training, and other unit events in a month view.

<Note>Screenshot placeholder: UNIT CALENDAR month view with event dots</Note>

## Open

Sidebar → **Calendar** (or **Unit Calendar**).

## Layout

**UNIT CALENDAR** header with:

- Month navigation (**←** / **→**) — changing months reloads events for that month
- **TODAY** — jump to the current month and day
- A thin loading bar while events are fetched
- Month grid (Sun–Sat) with colored dots for days that have events
- **Upcoming events** list (next items after today)

On compact screens, **Filters** opens a bottom sheet with search, date range, and event types. On larger layouts, filters appear in a right sidebar.

<Note>Screenshot placeholder: Desktop calendar with sidebar filters and upcoming list</Note>

## Live vs training mode

| Mode | What you see |
|------|----------------|
| **Live (sandbox or production)** | Events from your agency’s Cadmus Ops Calendar — the same scheduling data dispatch uses. Visibility follows your role (agency-private, shared dispatch, or org-wide events you are allowed to see). |
| **Training** | Sample events for practice. These are not live schedule data. |

If the calendar cannot load live events, the screen shows **Could not load calendar events**. If your agency has a very busy month and the server returns a partial list, a warning may appear asking you to narrow the date range or contact dispatch.

## Event types

Toggle which types appear on the calendar:

| Type | Typical use |
|------|-------------|
| **Shift** | Scheduled patrol or duty shifts |
| **Court** | Court appearances |
| **Call** | Scheduled callbacks or follow-ups |
| **Training** | In-service or mandated training |
| **PTO** | Planned time off |
| **Incident** | Follow-up or case-related dates |

<Note>Screenshot placeholder: EVENT TYPES filter checkboxes</Note>

## Filters

- **Search events…** — match title, description, or case number
- **DATE RANGE** — **FROM** / **TO** pickers
- Event type toggles above

Days with more than three events show **+N more** on the grid cell.

## Event details

Tap an event in the grid or **Upcoming events** list to open details:

- Title and type
- Start / end date and time
- Location and description when provided
- Case number or priority when applicable

<Note>Screenshot placeholder: Event detail modal</Note>

## Tips

- Filter to **Court** and **Training** at the start of the month for planning.
- Use **TODAY** after navigating away during a long shift.
- Pair with [Unit status](/mobile/unit-status) — calendar shows planned time; status reflects what you are doing now.

## Warnings

<Warning>Calendar is informational — official timekeeping may use a separate HR or scheduling system.</Warning>
